# AI & Data Science Portfolio

A modern, responsive portfolio website built with React, Bootstrap 5, and smooth animations.

## Features

- Responsive design with Bootstrap 5
- Smooth scroll animations using AOS
- Interactive sections: Hero, About, Skills, Projects, Education, Contact
- Modern UI with gradient effects
- Contact form integration

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

### Build for Production
```bash
npm run build
```

## Customization

Edit the content in the component files to personalize:
- `src/components/Hero.js` - Your name and introduction
- `src/components/About.js` - About you section
- `src/components/Skills.js` - Your technical skills
- `src/components/Projects.js` - Your projects
- `src/components/Education.js` - Your educational background
- `src/components/Contact.js` - Contact information

## Technologies Used

- React 18
- Bootstrap 5
- AOS (Animate On Scroll)
- React Icons
- EmailJS
